# RecipeBook 🍳

A modern recipe management application designed to create a clean, distraction-free cooking experience.

Built with React, TypeScript, and Tailwind CSS, RecipeBook helps users save, organize, and cook recipes effortlessly — whether imported from the web or added manually.


## ✨ Why I Built This

Most recipe websites are cluttered with ads, long stories, and poor mobile experiences.  
RecipeBook focuses on simplicity and usability by providing:

- Clean recipe reading
- Distraction-free cooking mode
- Smart recipe importing
- Fast and responsive experience


## 🚀 Features

- 📝 Add recipes manually
- 🔗 Import recipes from URLs
- 🔍 Smart recipe parsing
- 👨‍🍳 Distraction-free cooking mode
- ❤️ Favourite recipes
- 📱 Responsive across all devices
- 🗑️ Multi-select bulk delete
- 🎨 Modern UI with Tailwind CSS

## 📦 Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or pnpm

### Installation

```bash
# Clone the repository
git clone https://github.com/sajanbodele/RecipeBook.git

# Navigate to the project directory
cd RecipeBook

# Install dependencies
npm install

# Start development server
npm run dev
```

The app will be available at:

```bash
http://localhost:5173
```

## 🍽️ How to Use

### 1. Add a Recipe

You can:
- Paste a recipe URL for automatic parsing
- Add recipe details manually

### 2. Browse Recipes

View all saved recipes inside **My Recipes**.

### 3. Cooking Mode

Open any recipe to enter distraction-free cooking mode with:
- Step-by-step instructions
- Ingredient checklist
- Clean reading interface

### 4. Manage Recipes

- Favourite recipes using the heart icon
- Select multiple recipes for bulk delete
- Filter favourite recipes
